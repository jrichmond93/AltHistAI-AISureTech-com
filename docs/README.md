## 🛠️ Technical Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Build Tools**: None (static site)
- **Server**: Python HTTP Server (development)
- **Browser Support**: Modern browsers (Chrome 70+, Firefox 65+, Safari 12+, Edge 79+)
